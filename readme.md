各个配置文件作用介绍

Info.List
此文件在于给您下载站的文件添加简洁，如果未添加默认为‘不解释，你应该懂的，2333333’，添加格式如下
#文件名|!|#文件介绍|!|#图片设定|!|#附加介绍|!|#其他地址
#如果此项为空请设定值为|!|
#分割符为'|!|'

Passwd.Info
此文件为下载密码
#文件/文件夹途径|!|#密码

About.Info
此文件为关于页面介绍，您可以自行修改
